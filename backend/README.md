# MovieFlix Backend

## To Run Backend

- Refer `.env.sample` file and create `.env` file in root folder.
- Now Copy the content from `.env.sample` to `.env`
- Set ``` MONGO_URI, PORT, OMDB_API_KEY, JWT_SECRET```
- ``` OMDB_API_KEY ``` key can be generated from <a href="https://omdbapi.com/" target="_blank">OMDB API</a>
- ``` JWT_SECRET ``` key can be generated from <a href="https://www.jwt.io/" target="_blank">JWT.io</a>
- Once all this are done. Run ``` npm install ```
- After all the dependency get install run ``` npm start ```
